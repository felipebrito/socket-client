INSTRUÇÕES PARA ADICIONAR SEU LOGO PERSONALIZADO

1. Coloque sua imagem de logo com o nome "aparatoLogo.png" nesta pasta (Assets/Resources/Logo/).
   - Recomendamos uma imagem PNG com transparência 
   - Tamanho ideal: entre 128x128 e 256x256 pixels
   - Formatos suportados: .png, .jpg, .tga

2. A imagem será automaticamente detectada e exibida em todas as janelas de editor personalizadas.

3. Se você quiser atualizar a imagem, basta substituir o arquivo existente. O Unity recarregará automaticamente.

Observação:
- A imagem será dimensionada automaticamente para caber no espaço reservado em cada janela
- Para melhor visualização, use uma imagem com boa resolução e transparência
